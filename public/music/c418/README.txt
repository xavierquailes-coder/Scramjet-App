Put your legally owned C418 / Minecraft soundtrack audio files in this folder.
The jukebox looks for these filenames, for example:
Minecraft.mp3
Sweden.mp3
Wet Hands.mp3
Subwoofer Lullaby.mp3
Mice on Venus.mp3
Cat.mp3
13.mp3

The zip includes the jukebox/player code only, not the copyrighted songs.
